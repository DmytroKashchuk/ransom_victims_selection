Factor Analysis — Tech Co-occurrence

Reduce a sites × technologies matrix into latent tech stacks using NMF or Truncated SVD. Useful when you want to feed a logistic regression with a few interpretable components instead of thousands of dummy variables.

Workflow: 1) Upload CSV → 2) Choose data format (long or wide) and pick columns → 3) Build the binary matrix → 4) Run NMF / SVD → 5) Inspect components or download site × component scores.
Long format: one row per (site, tech) pair, e.g. columns site_url, technology.
Wide format: one row per site, with one 0/1 column per technology.
